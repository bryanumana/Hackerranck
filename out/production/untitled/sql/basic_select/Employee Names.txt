Write a query that prints a list of employee names (i.e.: the name attribute)
from the Employee table in alphabetical order.

The Employee table containing employee data for a company is described as follows:
| Column      | Type    |
| employee_id | Integer |
| name        | String  |
| months      | Integer |
| salary      | Integer |

SOLUTION:
SELECT NAME
FROM EMPLOYEE
ORDER BY NAME ASC
